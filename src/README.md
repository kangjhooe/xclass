# Source Code - Backend

**Version: 0.1**

Folder ini berisi source code backend aplikasi CLASS yang dibangun dengan NestJS.

## Struktur

- `modules/` - Modul-modul fitur aplikasi (auth, students, teachers, dll)
- `common/` - Komponen umum yang digunakan di seluruh aplikasi
- `migrations/` - Database migration files
- `main.ts` - Entry point aplikasi
- `app.module.ts` - Root module aplikasi

## Teknologi

- **NestJS** - Framework Node.js
- **TypeORM** - ORM untuk database
- **MySQL** - Database
- **JWT** - Authentication
- **TypeScript** - Programming language
