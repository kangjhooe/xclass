# Storage

**Version: 0.1**

Folder ini digunakan untuk menyimpan file-file yang diupload oleh pengguna aplikasi.

## Struktur

- `app/` - File-file aplikasi yang diupload

## Catatan

- Folder ini digunakan untuk file storage lokal
- Pastikan folder ini memiliki permission write yang sesuai
- Untuk production, pertimbangkan menggunakan cloud storage (S3, dll)
